# Collatz Stained Glass
**2進対数らせん空間における Voronoi 分割の適用**

**Author**: Hiroshi Harada  
**Date**: April 6, 2026  

---

## 概要 (Overview)
本研究は、数論における未解決問題「Collatz 予想（3n+1 問題）」の背後に潜む構造的秩序を、**2 進対数らせん空間（binary logarithmic spiral plane）**と **Voronoi 分割**を組み合わせることで可視化する、新しい数理可視化フレームワークを提案するものです。

自然数を 2 進的な幾何空間へ写像し、その上で Voronoi セルを生成することで、**逆 Collatz 木（preimage tree）を構成する自然数集合が、対数領域において隙間なく密充填されていく様子（Hausdorff 次元 $D_H \approx 1.0$）**を、発展ステップ数に応じた色彩変化とともに「ステンドグラス」として描き出します。

---

## 数学的・技術的アプローチ

### 1. 2進対数らせん空間への写像
自然数 $n$ を以下の極座標へ写像します：

- $r = \log_2(n)$  
- $\theta = 2\pi(r \bmod 1)$

この写像により、  
**2 進的自己相似性、偶奇による軌道の周期性、そして「1」への収束方向の一意性**が幾何学的に表現されます。

---

### 2. Voronoi 分割による密充填構造の抽出
$N = 3000$ までの母点を生成し、外周にダミー点を配置して無限セルを閉じ、その内部から $N = 1000$ の Voronoi セルを抽出します。

これにより、自然数の幾何学的配置が **閉じたタイル構造として安定化**し、**Perfect Packing に近い密充填構造**を視覚的に提示します。

---

### 3. 発展ステップ数による動的着色
Total Stopping Time に基づき、各セルを以下の三相グラデーションで着色します：

- **Cyan（早期）**：高速に発展する 2 の冪系列（主幹）  
- **Yellow（中期）**：ジャンプ軌道を経て到達する奇数群（5、13、17 など）や、$4m+1$ で表される無限系列に属する奇数  
  （例：5 系列 = 21 → 85 → 341 → …、13 系列 = 53 → 213 → 853 → …）  
- **Magenta（後期）**：長い軌道を経て空間を埋める深層ノード（27、31、41 など）

---

### 4. 因果の軌道（Causal Tree）のオーバーレイ
最終フレームでは、逆 Collatz 木の因果構造をらせん空間上に重ね、  
**各自然数がどの位置に到達するか、そこから偶数倍の枝がどのように伸びるか**を視覚的に示します。

## 実行環境 (Requirements)
本プログラムを実行するには、以下の Python ライブラリが必要です：

```bash
pip install numpy scipy matplotlib
```

---

## 使い方 (Usage)
1. スクリプトを実行します。
```bash
python collatz_stained_glass.py
```
2. 実行中、ターミナルに進捗（フレームのレンダリング状況）が表示されます。
3. 処理が完了すると、作業ディレクトリにアニメーション GIF ファイルが生成され、最後にすべてのセルが着色された「完成図」が別ウィンドウで表示されます。
